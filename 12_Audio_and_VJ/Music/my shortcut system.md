# My shortcut system

I use custom shortcuts/scripts to switch Windows power plans quickly before live sessions.

## Power mode targets

- Performance mode for real-time/live work.
- Balanced or saver mode for normal daily use.

## Ultimate power-plan baseline

Use these values for the live-performance profile:

1. Turn off hard disk after: `0 minutes` (`Never`)
2. PCI Express -> Link State Power Management: `Off`
3. Processor power management:
   - Minimum processor state: `100%`
   - Maximum processor state: `100%`
